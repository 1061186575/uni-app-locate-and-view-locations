有两种保存位置的json数据格式  

一种保存所有位置信息:  
    一个用户名对应多条位置信息  
    数据格式:  
        {  
        "shasha": [{"userName":"小王"}, {} ],  
        "zp": [{}],  
        }  


一种保存最近的位置信息:  
    位置信息里需要用户名  
    数据格式:  
        [{"errMsg":"getLocation:fail","userName":"小王","time":"2020-02-05 15:22:19"}, {}, {}]
